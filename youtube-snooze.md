Turn your idle screen into a window to peaceful video ambience. YouTube Snooze plays curated ambient videos - from cozy fireplaces to forest streams - creating the perfect backdrop for relaxation or background atmosphere.

## 📥 Installation

### Download from Microsoft Store (Recommended)
Get YouTube Snooze with automatic updates from the [Microsoft Store](https://apps.microsoft.com/store/detail/youtube-snooze/9NXXXXXX).

### Manual Installation
1. Download the latest `.scr` file from [Releases](https://github.com/cybesis/screensavers/releases)
2. Right-click the file and select **Install**
3. Open Windows Screen Saver settings
4. Select "YouTube Snooze" from the dropdown
5. Click **Settings** to customize

## ⚙️ Configuration

Access settings by right-clicking on YouTube Snooze in Screen Saver settings:

### Video Selection
- **Categories**: Enable/disable specific video types
  - ✓ Fireplace & Cozy Scenes
  - ✓ Nature Ambience
  - ✓ Ocean & Water
  - ✓ Rain & Storms
  - ✓ Forest & Wildlife
  - ✓ Space & Stars
  - ✓ Aquariums
  - ✓ City Ambience
  - ✓ Lo-Fi Music
  
- **Video Length**: Short (5-15 min), Medium (15-45 min), Long (45+ min), or Any

### Playback Options
- **Volume**: 0-100% or Mute
- **Shuffle**: Random order or sequential
- **Repeat**: Loop single video or rotate through playlist
- **Quality**: Auto (adaptive), 720p, 1080p
- **Start Delay**: Wait 0-60 seconds before first video

### Advanced Settings
- **Network Timeout**: How long to wait for video load
- **Error Handling**: Skip on error or retry
- **Offline Fallback**: Show static image when no connection
- **Resume Behavior**: Continue last video or start fresh
- **Clock Overlay**: Show/hide digital clock

### Pro Features
- **Custom Playlists**: Add your own YouTube video URLs
- **Blacklist**: Exclude specific videos you don't want
- **Schedule**: Different playlists for different times
- **History**: See what's been played recently
- **Favorites**: Mark videos for more frequent play

## 🔧 Troubleshooting

**"Error 153: Video player configuration error"**
- This has been fixed in v1.0.12 and later
- Update to the latest version
- Ensure WebView2 is installed

**Videos won't play / Black screen**
- Check internet connection
- Verify YouTube isn't blocked by firewall
- Try different video category
- Install [WebView2 Runtime](https://go.microsoft.com/fwlink/p/?LinkId=2124703)

**Video stuttering or buffering**
- Lower quality setting to 720p or Auto
- Enable Low Bandwidth Mode
- Check network speed
- Close bandwidth-heavy applications

**No audio**
- Check volume slider in settings (not muted)
- Verify Windows volume mixer
- Ensure audio device is working

**Screensaver exits immediately**
- Check Windows Screen Saver settings (wait time)
- Disable "On resume, display logon screen" if testing
- Ensure no mouse/keyboard activity
