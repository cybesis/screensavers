# Digital Rain Matrix Screensaver

Transform your idle screen into the iconic Matrix digital rain effect. Watch cascading green characters flow across your display with smooth, customizable animation.

**Developed by [Cybesis Studios](https://cybesis.com) — an indie solo developer**

---

## ✨ Features

### Visual Customization
- **Multiple Character Sets**: Katakana (classic Matrix), Binary, Digits, Symbols, Hex, ASCII, Cyrillic, Greek, Runic, and Emoji
- **Preset Configurations**: Choose from Movie Accurate, Classic, Performance, or Minimal presets
- **Rain Direction**: Down, Up, Left, or Right - watch the Matrix flow in any direction
- **Speed & Density Control**: Adjust how fast and thick the rain appears
- **Brightness Settings**: Fine-tune the glow intensity for your preference

### Interactive Elements
- **Mouse Ripple Effects**: Optional ripple distortion that follows your cursor (Pro)
- **Custom Messages**: Inject your own text into the rain stream (Pro)
- **Digital Clock**: Built-in clock overlay with customizable position
- **HTML Overlays**: Display custom content over the animation (Pro)

### Performance & Compatibility
- **GPU-Accelerated**: Smooth 60 FPS animation using optimized Canvas rendering
- **Multi-Monitor Support**: Works seamlessly across multiple displays
- **Lightweight**: Minimal memory footprint, runs efficiently on modern PCs
- **Offline Capable**: No internet connection required

## � Installation

### Download from the Microsoft Store
Get the latest version with automatic updates from the [Microsoft Store](https://apps.microsoft.com/detail/9ND4MDXV97GS).

After installing:
1. Open Windows Screen Saver Settings (search "screen saver" in Start menu)
2. Select "Digital Rain" from the dropdown
3. Click **Settings** to customize
4. Set your preferred wait time and click **Apply**

## ⚙️ Configuration

Access settings by right-clicking on the installed screensaver in Screen Saver settings, or click the **Settings** button.

### Quick Presets
Choose from ready-made configurations:
- **Movie Accurate**: Classic Matrix movie appearance

- **Performance**: Optimized for lower-end systems

### Customization Options
- **Animation Speed**: Control how fast the rain falls (0.1x to 3.0x)

## 🔧 Troubleshooting

**Screensaver doesn't start**
- Ensure you've set the wait time in Windows Screen Saver settings
- Check that "On resume, display logon screen" isn't blocking activation

**Performance issues / Low FPS**
- Lower the Frame Rate setting to 30 FPS
- Reduce Density to create fewer columns
- Try the "Performance" preset
- Update your graphics drivers

**Settings not saving**
- Run as Administrator once to create configuration files
- Check folder permissions for `%LOCALAPPDATA%\CybesisSaver`

**Black screen instead of animation**
- Install [Microsoft Edge WebView2 Runtime](https://go.microsoft.com/fwlink/p/?LinkId=2124703) (usually pre-installed on Windows 10/11)
