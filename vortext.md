# Vortext Screensaver

Dive into a hypnotic ASCII grid vortex inspired by Midjourney's iconic warp animation. Watch as characters swirl, stretch, and warp in an endless tunnel of mesmerizing patterns.

**Developed by [Cybesis Studios](https://cybesis.com)**

---

## ✨ Features

### Visual Effects
- **Vortex Warp Animation**: Hypnotic tunnel effect with grid distortion
- **ASCII Art Style**: Retro terminal aesthetic with modern smoothness
- **Dynamic Patterns**: Ever-changing geometric formations
- **Color Gradients**: Smooth color transitions across the vortex
- **Depth Perception**: Realistic 3D tunnel illusion

### Customization
- **Warp Speed**: Control how fast you travel through the vortex
- **Grid Density**: Adjust character spacing and pattern complexity
- **Color Schemes**: Multiple palettes (Monochrome, Rainbow, Neon, Cyberpunk)
- **Character Sets**: Choose ASCII symbols, Unicode blocks, or custom characters
- **Rotation**: Enable/disable spiral rotation effect
- **Pulse Effects**: Rhythmic expansion/contraction of the vortex

### Performance
- **GPU-Accelerated**: Smooth 60 FPS rendering
- **Lightweight**: Minimal resource usage
- **Responsive**: Adapts to any screen resolution
- **Multi-Monitor**: Synchronized across displays

## 📥 Installation

### Download from Microsoft Store (Recommended)
Get Vortext with automatic updates from the [Microsoft Store](https://apps.microsoft.com/store/detail/vortext-screensaver/9NXXXXXX).

### Manual Installation
1. Download the latest `.scr` file from [Releases](https://github.com/cybesis/screensavers/releases)
2. Right-click the file and select **Install**
3. Open Windows Screen Saver settings
4. Select "Vortext" from the dropdown
5. Click **Settings** to customize

## ⚙️ Configuration

Access settings by right-clicking on Vortext in Screen Saver settings:

### Animation Settings
- **Warp Speed**: Slow, Medium, Fast, or Ludicrous
- **Grid Size**: Fine, Medium, or Coarse character spacing
- **Rotation Speed**: Control spiral motion intensity
- **Depth**: Adjust how far the tunnel extends
- **Frame Rate**: 30 or 60 FPS

### Visual Style
- **Color Scheme**: Choose from pre-defined palettes
  - **Monochrome**: Classic black & white
  - **Matrix Green**: Digital rain inspired
  - **Neon**: Vibrant pinks and blues
  - **Cyberpunk**: Yellow and magenta
  - **Rainbow**: Full spectrum cycling
  - **Custom**: Define your own colors (Pro)
  
- **Character Set**: 
  - **ASCII**: Traditional terminal characters
  - **Blocks**: Unicode block elements
  - **Symbols**: Mathematical and special symbols
  - **Custom**: Use your own character set (Pro)

### Effects
- **Pulse Effect**: Rhythmic vortex breathing
- **Chromatic Aberration**: RGB split for retro VHS look
- **Scanlines**: CRT monitor effect overlay
- **Glow**: Enhanced luminosity on characters
- **Trail Effect**: Motion blur for smoother appearance

## 🔧 Troubleshooting

**Vortex animation stuttering**
- Lower frame rate to 30 FPS
- Reduce grid density
- Update graphics drivers

**Characters appear blurry**
- Disable "Trail Effect" in settings
- Increase grid size for larger characters
- Check display scaling settings

**Colors not displaying correctly**
- Try a different color scheme
- Check monitor color profile
- Ensure WebView2 is up to date

**Screensaver won't activate**
- Set wait time in Windows Screen Saver settings
- Check power management settings
- Ensure no fullscreen apps are running
