# Aurora Screensaver

Bring the mesmerizing beauty of the Northern Lights to your desktop. Aurora creates stunning, realistic aurora borealis animations with fluid wave patterns, twinkling stars, and dynamic color transitions.

**Developed by [Cybesis Studios](https://cybesis.com)**

---

## ✨ Features

### Visual Elements
- **Realistic Aurora Waves**: Fluid, organic light waves that flow across your screen
- **Dynamic Starfield**: Twinkling stars that add depth and atmosphere
- **Color Themes**: Multiple aurora palettes including Classic Green, Arctic Blue, Sunset Pink, and more
- **Smooth Transitions**: Gradual color shifts create an ever-changing display
- **Parallax Effects**: Multiple layers create depth and dimension

### Customization
- **Wave Speed**: Control the flow rate of the aurora waves
- **Color Intensity**: Adjust brightness and saturation levels
- **Star Density**: Choose how many stars fill the sky
- **Blend Modes**: Different rendering styles for unique effects
- **Movement Patterns**: Customize the aurora's behavior and motion

### Performance
- **GPU-Accelerated**: Smooth 60 FPS animation
- **Lightweight**: Minimal CPU usage and memory footprint
- **Multi-Monitor**: Seamless support for multiple displays
- **Offline**: No internet connection required

## 💻 System Requirements

- **Operating System**: Windows 10 (version 1809 or later) or Windows 11
- **Processor**: Any modern CPU (Intel/AMD)

## 📥 Installation

### Download from Microsoft Store (Recommended)
Get Aurora with automatic updates from the [Microsoft Store](https://apps.microsoft.com/store/detail/aurora-screensaver/9NXXXXXX).

### Manual Installation
1. Download the latest `.scr` file from [Releases](https://github.com/cybesis/screensavers/releases)
2. Right-click the file and select **Install**
3. Open Windows Screen Saver settings
4. Select "Aurora" from the dropdown
5. Click **Settings** to customize

---

## ⚙️ Configuration

Right-click on Aurora in Screen Saver settings or click **Settings** to customize:

### Quick Options
- **Aurora Type**: Choose from various aurora styles (Curtain, Arc, Corona)
- **Color Theme**: Select from pre-defined color palettes
- **Animation Speed**: Slow, Medium, Fast, or Custom
- **Star Effect**: Enable/disable the starfield background

### Advanced Settings
- **Wave Height**: Control the vertical span of aurora waves
- **Wave Frequency**: Adjust wave pattern complexity
- **Color Shift Rate**: How quickly colors transition
- **Brightness**: Overall luminosity control
- **Contrast**: Enhance or soften the effect
- **Frame Rate**: 30 or 60 FPS

## 🔧 Troubleshooting

**Aurora appears choppy**
- Lower the frame rate to 30 FPS
- Update your graphics drivers
- Reduce wave frequency complexity

**Colors look washed out**
- Increase Brightness and Contrast settings
- Try a different color theme
- Check monitor calibration

**Screensaver doesn't activate**
- Verify wait time in Windows Screen Saver settings
- Ensure no applications are preventing activation

**WebView2 error**
- Download [Microsoft Edge WebView2 Runtime](https://go.microsoft.com/fwlink/p/?LinkId=2124703)
