# Fireside Cats Screensaver

Transform your screen into a cozy fireplace scene with adorable pixel art cats lounging by the warm glow. Perfect for creating a relaxing ambiance while your computer idles.

**Developed by [Cybesis Studios](https://cybesis.com)**

## ✨ Features

### Cozy Atmosphere
- **Animated Fireplace**: Realistic flickering flames with ember particles
- **Adorable Cats**: Multiple cat characters with unique animations
- **Warm Lighting**: Dynamic fire glow that illuminates the scene
- **Ambient Sounds**: Optional crackling fire audio (Pro)
- **Seasonal Themes**: Holiday decorations and seasonal variations (Pro)

### Cat Behaviors
- **Sleeping Cats**: Peaceful resting animations
- **Playful Moments**: Occasional stretching, yawning, and movement

- **Personality Traits**: Each cat has unique characteristics (Pro)

### Visual Customization
- **Fire Intensity**: Adjust flame height and brightness
- **Room Lighting**: Control the ambient glow
- **Cat Selection**: Choose which cats appear
- **Background Options**: Different room styles and decorations (Pro)
- **Animation Speed**: Slow, relaxed pace or more active

### Interactive Elements
- **Clock Display**: Optional digital clock overlay
- **Weather Integration**: Show current temperature (Pro)
- **Custom Messages**: Add personal text to the scene (Pro)

## 📥 Installation

### Download from Microsoft Store (Recommended)
Get Fireside Cats with automatic updates from the [Microsoft Store](https://apps.microsoft.com/store/detail/XPFH5X6V21H5M1).

### Manual Installation
1. Download the latest `.scr` file from [Releases](https://github.com/cybesis/screensavers/releases)
2. Right-click the file and select **Install**
3. Open Windows Screen Saver settings
4. Select "Fireside Cats" from the dropdown
5. Click **Settings** to customize

## 🔧 Troubleshooting

**No sound effects**
- Ensure audio is enabled in settings
- Check Windows volume mixer
- Pro license required for sound features

**Cats not animating**
- Try increasing animation speed
- Restart the screensaver
- Update graphics drivers

**Fire looks choppy**
- Lower animation quality in settings
- Update to latest version
- Check for background processes using GPU

**Settings not saving**
- Run as Administrator once
- Check folder permissions for `%LOCALAPPDATA%\CybesisSaver`
